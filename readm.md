# IA Agents Course — Ejercicios y Proyectos

Repositorio personal para documentar y versionar **ejercicios**, **pruebas de concepto** y **proyectos derivados** del curso.

> Nota: algunos contenidos pueden estar alineados con el curso siguiente **@joanby/agents**.

## Objetivos

- Consolidar lo aprendido mediante prácticas reproducibles.
- Mantener un historial claro de decisiones técnicas y evolución del código.
- Desarrollar pequeños proyectos "de ingeniería" (con estructura, pruebas, documentación y CI cuando aplique).

## Temática / Qué encontrarás aquí

Según vaya avanzando, iré subiendo materiales como:

- **Fundamentos de agentes**: planificación, herramientas, memoria y orquestación.
- **Integraciones**: APIs, web scraping, bases de datos y pipelines.
- **Buenas prácticas de ingeniería**:
  - estructura de proyecto y modularidad
  - tipado/linting/formatting
  - tests (unitarios e integración)
  - logging y trazabilidad
  - configuración y secretos (dotenv, etc.)
  - automatización (Makefile/scripts)
  - CI/CD (GitHub Actions)
- **Proyectos personales**: experimentos y aplicaciones pequeñas orientadas a resolver problemas reales.

## Estructura (orientativa)

> La estructura puede cambiar a medida que crezca el repositorio.

- `exercises/` — ejercicios del curso
- `projects/` — proyectos personales derivados
- `notes/` — apuntes y conceptos clave
- `scripts/` — utilidades y automatización

## Cómo ejecutar (pendiente)

Añadiré instrucciones específicas por proyecto (requisitos, instalación y ejecución) conforme se incorporen.
